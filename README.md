# Bisection method

A project on the [bisection method](https://en.wikipedia.org/wiki/Bisection_method).

## Includes

- Slides made with LaTeX and [BEAMER](https://www.beamer.plus/home.html) under `slides/`;
- A simple Python implementation as a console application under `bisection_method/` that allows dynamically specifying the bounds, the function and the approximation precision.
